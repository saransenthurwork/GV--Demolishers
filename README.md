# GV Demolishers Website
Production-ready business website for GV Demolishers, built with Vite + React and deployed on Netlify with custom domain integration.

![Status](https://img.shields.io/badge/status-production-success?style=for-the-badge)
![Tech](https://img.shields.io/badge/Vite-React-purple?style=for-the-badge)

🌐 Live Site: https://gvdemolishers.com

---

## 🧩 Problem

The client needed a fast, modern, and responsive website to showcase demolition services and establish an online presence.

## 💡 Solution

Built a Single Page Application (SPA) using Vite + React, optimized for performance and deployed on Netlify with a custom domain configured via DNS.
---

## ⚙️ Tech Stack

* Vite
* React
* JavaScript (ES6+)
* CSS / Tailwind (if used)

---

## 🚀 Features

* Fully responsive design (mobile + desktop)
* Fast loading with optimized production build
* Clean and modern UI for business presentation
* Single Page Application (SPA) architecture
* SEO-friendly structure

---

## 🌐 Deployment

* Hosted on Netlify
* Custom domain configured via GoDaddy DNS
* HTTPS enabled with SSL (auto via Netlify)

---
